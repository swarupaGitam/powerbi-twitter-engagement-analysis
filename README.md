# Twitter Engagement Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes tweet engagement across different tweet categories by comparing URL clicks, user profile clicks, and hashtag clicks. The dashboard helps identify which types of tweets generate higher user interaction.

## Objective

To visualize and compare user interactions across tweet categories such as tweets containing media, links, and hashtags while applying specific filtering conditions.

## Key Features

* Clustered bar chart for interaction comparison
* Analysis of URL clicks, user profile clicks, and hashtag clicks
* Tweet category-based engagement insights
* Even-date tweet filtering
* Interaction-based filtering
* Word count filtering
* Time-based visualization logic using DAX

## Tools Used

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Microsoft Excel

## Files Included

* Twitter_Engagement_Analysis_Dashboard.pbix
* Twitter_Engagement_Dataset.xlsx
* Dashboard_Overview.png

## Dashboard Insights

The dashboard provides a comparative view of user engagement metrics across different tweet categories, helping identify content types that drive the most interaction.
